* 설치 방법
    * PostgreSql

```
$ brew update
$ brew install postgresql  
```

* postgreSql 시작 및 종료

```
$ Brew services start postgresql
$ Brew services stop postgresql
```

ps.혹시 role 관련해서 접속이 안된다면 구글링으로 해결해야 합니다.
